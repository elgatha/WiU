Project Name: WorldSurf

Project Description

The WorldSurf project was created to educate us in the many aspects of good user-centered design. During this project we will implement different styles, techiniques and usability processes to create a wireframe for a mobile application to represent WorldSurf. At the end of this month we should have a complete understanding of how a user-friendly application should be correctly laid out and how to properly utilize efficiently designed forms that will collect and send information to this application.
